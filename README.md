# STM32H7xx DriverLib

Trying to implement a driver library for Nucleo-H743ZI development board for educational purposes.



## The available resources

- #### Hardware Abstraction Layer

  - GPIO
  - SPI
  - ExtI
  - Interrupts



- #### External Peripherals

  - LED



- #### Examples

  - ##### Microcontroller

    - GPIO Blinky

    - GPIO Interrupt

    - SPI

  - ##### Peripherals

    - LED Blinky
